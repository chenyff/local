4、浮动元素会让出标准文档流的位置
 同一级兄弟元素，前面的元素浮动，后面的元素没有浮动，那么浮动元素会让出标准流的位置给后面的元素，视觉效果上有一个压盖效果，我们实际工作中不用浮动做压盖，我们用定位来制作压盖效果。
5、字围效果
  
浮动存在的问题
 1、浮动的元素不能撑开父级元素的高度
 2、浮动元素会影响后面的元素
   方法一：给父元素增加高度，有高度的盒子可以管住内部浮动元素，不受其他浮动元素影响
   解决了浮动元素之间的相互影响，没解决高度自适应
   方法二：clear:清除浮动，给受影响的盒子添加浮动属性
   属性值：left right both
   解决了浮动元素影响，没解决高度自适应，margin失效
   方法三：隔墙法
   1、外墙法 在两个盒子中间建一个有高度的墙，同时还需要清除浮动 
   <div></div>
   <div class="cl h20"></div>
   <div></div>
   解决了浮动元素影响，视觉上有了间距
   没解决高度自适应，margin的问题
   2、内墙法 在两个父元素底部增加一个元素 给这个元素clear:both
   虽然解决了所有的问题，但是不推荐使用，因为增加了很多无意义的标签，会使代码比较冗余
   方法四：overflow:hidden
   overflow 溢出处理方法
   属性值：auto hidden
   auto 溢出滚动 如果盒子的内容超出设置的高度会出现滚动条，显示的只有高度的部分，拉动滚动条才能查看全部内容
   hidden 溢出隐藏  超过盒子的部分自动隐藏 padding区域不隐藏 隐藏的是border外的内容
