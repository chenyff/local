jQuery 中的动画 animate() 
         语法：$(selector).animate({params},speed,callback); 
         params 必填参数 定义形成动画的css属性 
         speed 可选参数 规定效果的时长，他可以取以下值“slow”、“fast”或毫秒 
         callback可选参数 参数是完成动画后所执行的函数名称 
         如果把div元素移动到左边，直到left的值等于250为止 
         $("button").click(function(){ 
             $("div").animate({left:'250px'});
         }); 
         如需对位置进行操作，要记得首先把元素的 CSS position 属性设置为 relative、fixed 或 absolute！

         jquery animate  操作多个属性
         $("button").click(function(){ 
             $("div").animate({ 
                 left:'250px', 
                 opacity:'0.5', 
                 height:'150px', 
                 width:'150px' 
            }); 
         });
         注意：animate并不是可以操作所有属性，色彩动画并不包含在内，同时，像margin-left或者padding-top这样的属性，
         必须改成marginLeft或者paddingTop这种
         只有数字值可创建动画（比如 "margin:30px"）。字符串值无法创建动画（比如 "background-color:red"）
